# carwash

Создать мобильное приложение для сети автомоек

Требования:
 - Запись с выбором бокса, времени и оплатой
 - Возможны акции
 - Может быть личный кабинет
 - Роли: директор, бухгалтер, кассир, мойщик, клиент
 - Мойщик может видеть уведомление после оплаты
 - Возможна оплата через кассу или приложение
 - Заказ можно отменить до того когда автомобиль попадает в бокс
 - Форма обратной связи насчет качества услуги
 - Рейтинг мойщиков
 - Блэклист
 - VIP лист
 - Скидочная система для таксистов, спецслужб и т.д.
 - Оповещения
 - Статистика по нагрузке боксов, прогнозирование
 - Корсаков: 5 боксов, Долинск: 7 Боксов, Ю-С: 10 боксов
 - Средний срок мойки: 10-15 мин
 - Режим работы: 9-21
 - 1-2 бокса могут быть зарезервированы кассиром под живую очередь
 - Цена услуг увеличивается с габаритом
 - Пользователь может регистрировать свои машины
